# Beamer Template

这是一个学术性的LaTeX Beamer模板，优点是非常简洁，适用于多公式，多算法的应用场景，缺点是不非常灵活。你也可以根据自己的需要随意使用和修改这个模板。

# Usage

1. 确保你的电脑上安装了Ctex中文套装和TexStudio编辑器（两者均是开源软件）

2. clone项目
```bash
git clone https://github.com/SunYanCN/Latex-Beamer-Template
```
3. 使用Xelatex编译文档即可
